🚀 Installazione & Sviluppo

  1. Clona: git clone https://github.com/hannesmitterer/bioarchittetura.git
  2. Installa Jekyll & dipendenze: gem install bundler jekyll bundle install
  3. Avvia: bundle exec jekyll serve --livereload
  4. Apri: http://localhost:4000
Comandi Utili
  • Sviluppo: bundle exec jekyll serve --livereload --drafts
  • Build produzione: bundle exec jekyll build
  • Verifica: bundle exec jekyll doctor
  • Pulizia: bundle exec jekyll clean

I Nostri Valori

🌱 Sostenibilità Ambientale

Promuoviamo progetti architettonici che rispettano i cicli naturali e minimizzano l'impatto ambientale attraverso l'uso di materiali locali e rinnovabili.

🏠 Comfort Abitativo

Ricerchiamo soluzioni per migliorare la qualità della vita negli spazi abitativi, con particolare attenzione al benessere psico-fisico degli occupanti.

💡 Innovazione Tecnica

Integriamo tecnologie innovative con saggezza costruttiva tradizionale, creando un equilibrio tra progresso e rispetto per l'ambiente.
